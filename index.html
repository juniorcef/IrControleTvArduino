Projeto: Utilizando um controle remoto para acender leds

Nesse projeto iremos utilizar um controle remoto (pode ser aquele controle antigo que você não usa mais) para acender leds. No próximo projeto (semana que vem), iremos utilizar a mesma ideia desse projeto, porém, agora, para acender uma lâmpada (a mesma usada nas residências).
